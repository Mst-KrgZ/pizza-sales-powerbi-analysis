# 🍕 Pizza Sales Analysis – Power BI Dashboard

![Pizza Dashboard](images/Screenshot%202026-03-08%20151620.png)

This project analyzes one year of sales data from a fictional pizza restaurant using Power BI.

The goal is to explore sales patterns, customer preferences, and product performance to support data-driven business decisions.

---

# 📊 Dashboard Overview

The interactive Power BI dashboard provides insights into:

• Revenue performance  
• Best and worst selling pizzas  
• Category distribution  
• Hourly demand patterns  
• Seasonal trends  
• Pizza size performance

---

# 📈 Key Metrics

Total Revenue: $817,860  
Total Orders: 21,350  
Total Pizzas Sold: 50,000  
Average Order Value: $38.31  
Average Pizzas per Order: 2.32  

---

# 📂 Dataset

The dataset contains detailed order information including:

• Order date and time  
• Pizza category  
• Pizza size  
• Quantity ordered  
• Price per item  
• Ingredients
